Die folgende Schritte sind notwendig um die UI zum Laufen zu bekommen:
- aktuelle Version von node und npm installieren (https://nodejs.org/en/download/package-manager/)
- aktuelle Version von angular installieren (https://www.npmjs.com/package/angular2)
- mit einer shell in den folder wechseln, wo sich diese README Datei befindet
- npm install ausführen
- bower install ausführen
- npm start ausführen
- Die UI wird auf Port 3000 gestartet
